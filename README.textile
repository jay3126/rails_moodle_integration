*Set Up:*

_Set up the Rails app on your localhost:_
7.  $ git clone git://github.com/sorsaffari/rails-moodle.git
8.  $ cd rails-moodle
9.  $ bundle install
10. $ rails generate layout:install bootstrap3
11. $ rake db:migrate
12. $ rake db:seed
13. $ rails server

_Visit the app at http://localhost:3000_
